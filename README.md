# 🛒 Shopper Spectrum

## 📌 Project Overview

Shopper Spectrum is an end-to-end Data Analytics and Machine Learning project developed using the Online Retail dataset.

The project analyzes customer purchasing behavior using RFM Analysis and K-Means Clustering, provides personalized product recommendations, predicts customer segments, and presents interactive business dashboards through Streamlit.

---

## 🎯 Objectives

- Clean and preprocess retail transaction data.
- Perform Exploratory Data Analysis (EDA).
- Build RFM features.
- Segment customers using K-Means Clustering.
- Develop a Recommendation System.
- Predict customer segments.
- Build an interactive Streamlit dashboard.
- Generate business insights for decision-making.
